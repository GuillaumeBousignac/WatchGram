- Recurring Issues and Points of Confusion

Lack of chat functionality
All three testers mentioned the need for a more advanced chat system or a chat between users, highlighting a clear lack of interaction on the website.

No major blocking issues were reported.

Create or improve a chat system between users.

Fast understanding of the website’s purpose.
Generally smooth and intuitive navigation.
Design appreciated by all testers.
