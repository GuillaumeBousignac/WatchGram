Objectives, Assumptions, Risks, and Constraints

Measurable Objectives:
- Allow users to compare at least two watches based on six technical criteria
- Reduce the number of platforms used for watch searches
- Increase engagement (likes, favorites, comparisons)

---

Assumptions:
- Users appreciate a social media-inspired interface
- Simplified technical data facilitates decision-making
- Visual comparison improves understanding of differences

---

Risks:
- Limited user sample (students)
- Potentially incomplete watch data
- Indirect competition with existing platforms

---

Constraints:
- Project conducted within an academic framework
- Data not connected to real-world databases
- Web application without a persistent backend
- Functionality limited to the prototype
